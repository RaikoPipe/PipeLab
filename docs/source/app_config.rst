App Configuration
==================

Color and shape configuration used in FunctionTestApp. See `Tkinter Colors
<https://i0.wp.com/www.wikipython.com/wp-content/uploads/Color-chart-capture-082321.jpg?resize=1536%2C869&ssl=1>`_

Active colors: Used when mouse is hovered over element.
Message colors: Used for message output by the process message display.

Border colors of buttons are not customizable.
